# AI Agentic LinkedIn Post Architect

A high-performance, agentic workflow for creating professional LinkedIn content using CrewAI, FastAPI, and UV. This system orchestrates a 3-agent specialized team to research, write, and document LinkedIn content blueprints.

**Note:** This project is currently optimized for local LLMs via [LM Studio](https://lmstudio.ai/) (specifically `zai-org/glm-4.6v-flash` with a 8192+ context window).

## 🚀 Optimized Workflow (3-Agent Team)

1.  **Market Analyst**: Conducts deep-dive research into trends, keywords, and audience behavior. Defines the optimal posting window.
2.  **Executive Copywriter**: Transforms strategy into high-impact LinkedIn content. Handles drafting, SEO formatting, and internal quality control in a single pass.
3.  **Operations Manager**: Packages the final result into a professional **Content Blueprint PDF** document.

## 🛠️ Features

- **Blazing Fast**: Consolidated 3-step pipeline reduces token usage and processing time.
- **Professional Outputs**: Generates structured PDFs with Navy Blue themes, strategic rationales, and polished content blocks.
- **Resilient**: Multi-iteration (max_iter=3) logic handles transient LLM failures.
- **Clean Architecture**: Decoupled agents, tools, and storage for maximum stability.

## 💻 Tech Stack

- **CrewAI**: Multi-agent orchestration.
- **FastAPI**: Modern web layer.
- **UV**: Blazing fast package manager.
- **fpdf2**: Professional document generation.
- **LM Studio**: Local inference server.

## 🚦 Getting Started

### Prerequisites

- [LM Studio](https://lmstudio.ai/) running `glm-4.6v-flash` or similar.
- **Context Length**: Set to **8192** or higher in LM Studio.
- Serper API Key (for web search).

### Quick Start

1.  **Setup Environment**:
    ```bash
    uv install
    cp .env.example .env # Add your SERPER_API_KEY
    ```

2.  **Start API Server**:
    ```bash
    uv run main.py | fastapi dev
    ```

3.  **Run Fast Test**:
    ```bash
    uvicorn main:app --host 0.0.0.0 --port 8000
    ```

    ```bash
    uv run python verify_project.py
    ```
